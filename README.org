#+TITLE: Evil Guix Channel

This channel contains, unfortunately, proprietary software that cannot
be included in GNU Guix. Package modules live under the =evil=
namespace.

While I do not personally endorse proprietary software, and encourage
you to appeal to Libre Software, there is the unfortunate need for
proprietary software still.

* Packages

- =discord=: Discord's proprietary desktop client for x86_64 Linux.

* Add The Channel

Add this channel to =~/.config/guix/channels.scm=:

#+BEGIN_SRC scheme
(channel
 (name 'evilguix)
 (url "https://github.com/MMagueta/EvilGuix"))
#+END_SRC

Then run:

#+BEGIN_SRC sh
guix pull
guix install discord
#+END_SRC

The Discord package definition follows the stable Linux source
versions pinned by Nixpkgs.
